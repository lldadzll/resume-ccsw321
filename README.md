# Ahad Resume Website (CCSW321 Assignment #1)

This folder contains a simple student-style resume website.

## Files
- index.html
- cssweb.css  (main stylesheet required by assignment)
- style.css   (compatibility; imports cssweb.css)
- app.js      (small JS for demo contact form)
- assets/img/*.svg (project cover placeholders)

## Bonus
- Bonus #2 (Bootstrap): Navbar + Cards are used (Bootstrap 5 via CDN).
- Bonus #1 (GitHub Pages): publish this repo/pages and include the live link inside your PDF submission.

## Publishing to GitHub Pages (quick)
1) Create a new repository (or use an existing one).
2) Upload these files to the repo (root).
3) Go to Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main, folder: / (root)
4) Save, then wait for the link to appear.
